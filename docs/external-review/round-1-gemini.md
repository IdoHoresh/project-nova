## 2. The Affect Layer is a Cosmetic Wrapper Over a Geometric Heuristic

**Severity:** CRITICAL
**Dimension:** §6.2 (Academic/Scientific Critique)
**Finding:** The core product claim is that the cognitive architecture predicts failure via an affect model (Schultz RPE / Russell circumplex). This is empirically false. The `AffectState` directly hardcodes anxiety as a function of empty cells (`anxiety += 0.7 * max(0.0, (3 - empty_cells) / 3)`). It is not a cognitive prediction generated by an LLM; it is a linear mathematical inversion of free board space. The "RPE" is calculated via a trivial heuristic that counts horizontally adjacent equal-value tiles. The entire "Cognitive Architecture as Moat" is a high-cost semantic facade covering a basic algorithmic 2048 heuristic.
**Citation:** `nova_agent/affect/state.py` lines ~27-31; `nova_agent/affect/rpe.py` lines 4-13; `docs/decisions/0001-cognitive-architecture-as-product-moat.md`.

## 3. The Baseline Bot Control Group is Invalidated by Temperature Asymmetry

**Severity:** HIGH
**Dimension:** §6.5 (Experimental Design)
**Finding:** ADR-0007 introduces the Baseline Bot to prove the cognitive layer adds predictive lead time. However, Amendment 1 strictly pins the Bot's temperature to 0 while Carla remains at 0.7. The Bot is deterministically forced into a greedy failure path with zero variance, while Carla benefits from stochastic exploration. This completely breaks the "same intelligence" baseline rationale. It is not an ablation of the cognitive architecture; it is an unequal comparison between a greedy-deterministic policy and a stochastic-exploratory one, rendering the Δ ≥ 2 threshold unfalsifiable.
**Citation:** `docs/decisions/0007-blind-control-group-for-cliff-test.md` Amendment 1 §A1.3.

## 4. Explicit Linter Suppression to Swallow Exceptions

**Severity:** HIGH
**Dimension:** §6.10 (Technical Debt / Review Theatre)
**Finding:** The project claims rigorous quality control via automated gates and checklist reviews, yet the core loop systematically masks its own failures. The end-of-trial reflection step is wrapped in a bare except block that actively suppresses the Ruff blind-except rule (`except Exception: # noqa: BLE001 \n pass`). This is review theatre: tests pass because failures are purposefully buried.
**Citation:** `nova_agent/lab/cliff_test.py` line 471; `nova_agent/decision/baseline.py` line 64; `nova_agent/decision/tot.py` line 180; `LESSONS.md`.

## The single attack the author should not be allowed to dodge

The author must not be allowed to dodge the fact that anxiety is hardcoded to `(3 - empty_cells) / 3`. The entire methodology — State-Transition Signatures, the Russell circumplex integration, and the commercial claim of predicting human churn — relies on the LLM cognitive layer interpreting the board and feeling "anxiety" before a crash. The architecture is a multi-thousand-dollar Rube Goldberg machine wrapping a basic `empty_cells < 3` heuristic.
