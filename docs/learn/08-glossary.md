# 08 — Glossary

> Quick reference for jargon. Look things up here when they come up in the other docs.

## Terms in alphabetical order

### ADB (Android Debug Bridge)
A command-line tool that lets your computer control an Android device or emulator. Nova uses it to take screenshots and send swipes.

### Affect
The general term in psychology for "feelings" — including mood, emotion, and motivational states. In Nova, the affect system is the bundle of variables (valence, arousal, dopamine, frustration, anxiety, confidence) that influence her decisions.

### Agent
An AI system that perceives an environment and takes actions in it. Nova is an agent. She perceives the 2048 board and acts via swipes.

### Appraisal
The process of evaluating a situation in emotional terms. *"Is this good or bad? Did I expect it? Can I cope?"* In Nova's architecture, appraisal happens before each move — the system computes affect from the current situation, then verbalizes it for the VLM.

### Black-box agent
An agent that interacts with its environment only through external interfaces (screen, touch input) — no access to internal state, code, or memory. Nova is strictly black-box.

### Brain panel
The right column of the demo UI. Shows Nova's internal state (mood, memories, reasoning) in real time.

### Chunk
A unit of meaningful information held in working memory. Cognitive scientists estimate humans can hold ~4 chunks (Cowan 2001). For Nova, chunks are things like "current board," "recent move," "retrieved memory."

### CoALA (Cognitive Architectures for Language Agents)
A 2024 paper (Sumers et al.) that proposes a unified framework for building LLM agents based on cognitive architectures. Nova uses CoALA's vocabulary throughout.

### Cosine similarity
A way to measure how similar two embeddings (lists of numbers) are. Outputs a value between -1 and +1. Used in vector databases to find similar past states.

### Decision loop
The cycle Nova runs over and over: perceive → retrieve → appraise → decide → act → evaluate → write to memory → repeat.

### Dopamine
In real brains, a neurotransmitter that signals reward prediction error. In Nova, a software variable that spikes on better-than-expected outcomes and shapes emotion updates.

### Dual-process theory
The idea (Kahneman, Stanovich) that human cognition has two modes: a fast, intuitive "System 1" and a slow, deliberative "System 2." Nova's intuition mode (single VLM call) and deliberation mode (Tree of Thoughts) is a metaphor for this.

### Embedding
A list of numbers that represents the "meaning" of something (a board, a sentence, an image). Generated by a model. Used so we can search for similar things via math instead of exact text matches.

### Emulator
A program that pretends to be another device. Nova plays a 2048 Android app running on an Android emulator on the dev machine.

### Episodic memory
Memory of specific events with time and context. *"Two games ago I lost on a tight board."* One of the three Tulving memory types.

### Forgetting curve
The empirically observed pattern of how memories fade over time. Power-law shape (slow then fast then slow). Nova uses it to weight recency in retrieval.

### Generative Agents
A 2023 paper (Park et al.) that introduced the memory-stream + reflection architecture for LLM agents. Nova borrows their retrieval formula directly.

### Heuristic
A rule of thumb that simplifies decision-making. "Keep the largest tile in a corner" is a 2048 heuristic. Nova uses heuristics as a fallback when the VLM fails.

### Importance score
A 1–10 rating attached to each memory, indicating how memorable it is. Computed both programmatically and via an LLM rating, then averaged. Higher importance = more likely to surface in retrieval.

### LLM (Large Language Model)
A type of AI model trained on text. Examples: Claude, GPT, Gemini. Generates text given a prompt.

### Loss precondition
The board state that *led to* a loss, not the board state at the moment of game-over. When Nova loses catastrophically, the system flags the boards from 3–5 moves before death as trauma memories.

### Modular Agentic Planner (MAP)
A 2025 *Nature Communications* paper (Webb et al.) that builds a brain-inspired modular LLM agent for planning tasks. Cited as the flagship "brain-inspired LLM agent" reference.

### Persona
A defined player archetype with characteristic playstyle. Beginner, optimizer, fast-tapper, etc. v2 feature.

### Power-law decay
A mathematical curve used to model forgetting. Slower-than-exponential at long timescales. Better fit to human data than exponential decay.

### Procedural memory
Memory for skills and motor patterns. The reflexes you don't have to think about. In Nova, partly implicit in the VLM weights and partly in the heuristic fallback.

### Prompt
The text (and images) sent to a VLM/LLM as input. Nova builds a fresh prompt for each move, including the board, retrieved memories, and current affect description.

### ReAct (Reasoning + Acting)
A 2022 paper (Yao et al.) introducing the pattern of interleaving "thought" and "action" in an agent's prompt. Nova's default decision path uses ReAct.

### Reflection
The post-game step where Nova generates a verbal summary of what she did, what worked, and what failed. Stored as semantic memory rules. From Reflexion (Shinn et al. 2023) and Generative Agents (Park 2023).

### Reflexion
A 2023 paper (Shinn et al.) introducing verbal reinforcement learning — the agent reflects on its failures in language and stores those reflections for later attempts. Nova uses this for post-game reflection.

### Relevance
In retrieval scoring: how similar a stored memory is to the current situation, measured via cosine similarity of embeddings.

### Retrieval
The process of pulling memories out of long-term storage when needed. Nova's retrieval ranks memories by `α·recency + α·importance + α·relevance`.

### RPE (Reward Prediction Error)
The difference between actual reward and expected reward. `δ = actual − expected`. The Schultz/Dayan/Montague (1997) finding that this signal is encoded by dopamine in the brain. Nova's central reward signal.

### scrcpy
An open-source tool that mirrors an Android device's screen as a video stream. Nova uses it to feed a smooth live game video to the brain panel.

### Semantic memory
Memory for general facts and rules, decontextualized from specific events. *"Keeping the largest tile in a corner is generally good."*

### SQLite
A small file-based database. Used by Nova to store the structured columns of memory entries.

### Take-The-Best
A heuristic decision rule from Gigerenzer & Goldstein (1996): rank cues by importance, decide based on the first cue that discriminates between options. Nova uses Take-The-Best as a fallback move policy.

### TD-error (Temporal-Difference error)
A concept from reinforcement learning equivalent to RPE. The mathematical foundation for Schultz's dopamine theory.

### ToT (Tree of Thoughts)
A 2023 paper (Yao et al.) introducing deliberate search over branches of reasoning. Nova uses ToT in her deliberation mode — generates multiple candidate moves, evaluates each, picks the best.

### Trauma memory
A memory entry tagged as `trauma` because it precipitated a catastrophic loss. Has elevated importance and a wider similarity threshold for retrieval, biasing future caution.

### Valence
The "good vs bad" axis of emotional state. Positive valence = feeling good; negative = feeling bad.

### Vector database
A database that stores embeddings and supports similarity search (find the items most "near" a query). Nova uses LanceDB.

### Visualization
Generally, the brain panel and all the UI elements that make Nova's internal state legible to a viewer.

### VLM (Vision-Language Model)
An LLM that can also process images. Nova uses Claude with vision (or equivalent) as her decision-maker.

### Working memory
The small amount of information held actively for the current task. ~4 chunks for humans (Cowan 2001). For Nova, it's the active section of the prompt — current board, last few moves, retrieved memories, current affect.

### WebSocket
A real-time bidirectional connection between two programs over a network. Nova's agent (Python) and the brain panel (Next.js) talk over WebSocket so the panel updates the moment internal state changes.
